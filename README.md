# [Semana OmniStack 11.0](https://rocketseat.com.br/week/aulas/11.0)
Curso RocketSeat


## Backend
API criada em NodeJS.
Para rodar, executar:

### `npm install`
### `npm start`

O sistema rodará na porta 3333.

## Frontend
Aplicação web criada em ReactJS.
Para rodar, executar:

### `npm install`
### `npm start`

O sistema rodará na porta 3000.
